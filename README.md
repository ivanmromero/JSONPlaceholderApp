# JSONPlaceholderApp

## Descripción

La aplicación tiene como objetivo mostrar noticias y detalles asociados a ellas, así como tambien información de usuarios y sus ubicaciones

## Patrones de Diseño

### Model-View-ViewModel (MVVM)

Se aplica el patrón de arquitectura MVVM para separar de la lógica de la interfaz el manejo de datos y el modelo de estos. Para conseguir mantener el código organizado y facilitar las pruebas unitarias.

## Bibliotecas

- **SwiftLint**: Para aplicar convenciones de estilo y realizar análisis del código Swift.
- **XCTest**: Para pruebas unitarias en Swift.
- **WebKit**: Para visualizacion de webs.
